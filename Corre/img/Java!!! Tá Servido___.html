Sua resposta
É um elemento de estrutura que possibilita a criação de objetos e suas definições.
2- Defina Objeto
Sua resposta
É uma instância dentro de uma classe que contém dados.
3- O que é um método?
Sua resposta
É uma função ou um bloco de código que executa uma ação específica,
4- O que é um método Get dentro do JAVA?
Sua resposta
Método de gerenciamento para acessar o valor de um atributo, tornando o controle mais simplificado.
5- O que é um método Set dentro do JAVA?
Sua resposta
Usado para definir o valor de uma variável ou atributo em um objeto. É usado para garantir a encapsulação de dados em seus objetos, tornando seus atributos privados e acessando-os somente através de métodos públicos como set e get.
6- O que é um método Construtor dentro do JAVA?
Sua resposta
Em Java, um método construtor é um tipo especial de método usado para criar e inicializar objetos de uma classe
7- O que é um Método Principal dentro do JAVA?
Sua resposta
A classe que contém o método "main", um ponto de entrada para a execução de um programa e uma função especial que é invocada quando o programa é iniciado e é onde a execução do código começa.
8- Nos conceitos de POO fortemente usados no JAVA, o que significa um Encapsulamento?
Sua resposta
Encapsulamento refere-se ao princípio de esconder os detalhes internos de implementação de uma classe e fornecer uma interface pública para interagir com os objetos dessa classe.
9- Nos conceitos de POO fortemente usados no JAVA, o que significa o Polimorfismo?
Sua resposta
O polimorfismo é um princípio que permite que objetos de diferentes classes sejam tratados de maneira uniforme, ou seja, um objeto pode assumir várias formas.
10- Nos conceitos de POO fortemente usados no JAVA, o que significa Herança?
Sua resposta
Herança ocorre quando uma classe pode herdar características (atributos e métodos) de outra classe. A classe que herda é chamada de classe filha ou subclasse, e a classe da qual ela herda é chamada de classe pai, superclasse ou classe base.
11- Nos conceitos de POO fortemente usados no JAVA, o que significa Abstração?
Sua resposta
A abstração em Java permite criar representações de objetos do mundo real, concentrando-se em suas características essenciais e ignorando os detalhes menos importantes.
